# 👨‍💼 Employee Data Analysis Dashboard

> A beginner-friendly HR analytics project built using Excel/Google Sheets, demonstrating insights from employee data through pivot tables and interactive visualizations.

---

## 📌 Project Overview

This project analyzes employee data and presents key workforce insights through an interactive dashboard. The dashboard provides a clear visualization of employee distribution, salary trends, attendance patterns, and work mode preferences.

The goal of this project is to transform raw employee data into meaningful insights using spreadsheet tools and data visualization techniques.

---

## 🚀 Features

* 📊 Interactive dashboard using Pivot Tables
* 📈 Visual representation of employee data
* 🎯 Analysis based on:

  * Department-wise employee distribution
  * Gender distribution
  * Work Mode Analysis (Remote, Hybrid, On-Site)
  * Location-wise employee count
  * Attendance percentage analysis
* 📂 Clean and structured dataset

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel / Google Sheets
* Pivot Tables
* Data Visualization (Bar Charts, Pie Charts)
* Data Cleaning & Organization

---

## 📂 Dataset Information

* Total Records: **1,000 Employees**
* Fields Included:

  * Employee ID
  * Name
  * Age
  * Gender
  * Department
  * Job Title
  * Salary
  * Location
  * Work Mode
  * Attendance Percentage
  * Email Address
  * Date of Joining

## 📊 Key Insights

* The **IT Department** has the highest number of employees.
* Most employees prefer the **Hybrid** work mode.
* Employee attendance remains consistently above **85%**.
* Salary distribution varies significantly across departments.
* Major workforce concentration is observed in metropolitan cities.
